# ab-modal

## Install

With [npm](http://npmjs.org) do:

```bash
$ npm install ab-modal --save-dev
```

## Usage
	
	1 - git clone https://github.com/antoniobrandao/socialhub.git socialhub_test

	2 - cd socialhub_test

	3 - npm install

	4 - gulp watch

	5 - open http://localhost:3000/ in Browser

	6 - CTRL-C twice to exit
